# SpringBoot demo 交流学习

#### 介绍
SpringBoot示例

#### 软件架构
- SpringBoot 2.3.0.RELEASE
- Mybatis
- MySQL
- Mybatis Plus
- Freemarker
- Thymeleaf
- ...

#### 安装教程
1. maven命令：mvn package -Dmaven.test.skip=true
2. 导入eclipse或者Idea
3. 运行主启动类：XXXXApplication.java
